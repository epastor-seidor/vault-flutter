Configuración Servidor Proyectos

Alex Rivera
•
calendar_today 24 de Octubre, 2023

Esta guía detalla los pasos críticos realizados para configurar el clúster de producción. Se ha priorizado la seguridad mediante la implementación de claves SSH de 4096 bits y la desactivación del login root por contraseña.

"La automatización no es solo para ahorrar tiempo, es para garantizar la repetibilidad y eliminar el error humano en entornos de alta disponibilidad."

Fragmento de Configuración Nginx

server {
    listen 443 ssl http2;
    server_name core.atelier.digital;

    ssl_certificate /etc/letsencrypt/live/atelier/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/atelier/privkey.pem;

    location / {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
